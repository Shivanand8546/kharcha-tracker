# 💰 Kharcha Tracker

A modern **Expense Tracking and Personal Finance Management System** built using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**.

Kharcha Tracker helps users efficiently manage their daily income and expenses, monitor financial activities, and gain valuable insights through interactive dashboards and analytics.

# 📖 About the Project

Kharcha Tracker is a full-stack web application developed to simplify personal finance management. Users can securely register, log in, add income and expense transactions, categorize records, filter data by date, and analyze their financial activities through an interactive dashboard.

The application is designed with a modern, responsive user interface and follows the MERN Stack architecture to provide a fast, secure, and scalable experience.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Shivanand8546/Kharcha-Tracker.git
```

## Navigate to Project

```bash
cd Kharcha-Tracker
```

## Install Frontend

```bash
cd frontend
npm install
npm start
```

## Install Backend

```bash
cd ../backend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `config/config.env` file inside the backend folder.

```env
PORT=5000

MONGO_URL=YOUR_MONGODB_CONNECTION_STRING

JWT_SECRET=YOUR_SECRET_KEY
```
